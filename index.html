UAS Generator 2026 V2

Mac desktop application for generating Anthem MLTC workflow documents from UAS assessment CSV files.

Download for Mac

Setup Steps

  1. Download and unzip the file.
  2. Open the folder.
  3. Double-click UAS Generator.app.
  4. Run System Check first.
  5. Complete onboarding.
  6. Enter your email and license key.
  7. Select your CSV and generate documents.

Internet connection is required for license activation. User must review all generated documents before submission.
