How to Install and Activate

  1. Download the ZIP file using one of the buttons above.
  2. Open the ZIP file to extract the app.
  3. For Mac: open the extracted app.
  4. For Windows: keep the full extracted folder together and open UAS_Generator_Windows.exe.
  5. Enter your license key when prompted.
  6. Accept the Terms of Use.
  7. Run System Check if available.
  8. Run Test Generation if available.
  9. Start normal use.
